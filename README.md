# Score26

Application de pronostics pour la Coupe du Monde 2026.
Prédit les scores, suis tes résultats, partage tes performances.

## Stack
- Frontend : React + Vite + TailwindCSS (PWA)
- Backend : Node.js + Express
- BDD : SQLite

## Lancer le projet

# Backend
cd server && npm run dev

# Frontend
cd client && npm run dev
